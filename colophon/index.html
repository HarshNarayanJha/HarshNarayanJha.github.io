Colophon

A page about this website.

Tech Stack

This website is fully handcrafted using Astro, TailwindCSS , MDX , Svelte, and Tippy.js . The website's source code is fully open source on GitHub , which is also the place where this site is hosted .

Tool Stack

I was able to handcraft this website thanks to TypeScript — Powerful Type safety, Bun — A fast runtime and tooling, Zed — Next level code editor that works for me.

Typography

I use a very wide range of typography for this website Plein for big text, Satoshi for most of the headings, Switzer as the sans body font, Array for some special stuff. Architects Daughter for alternate stylistic text, and finally Monaspace Argon as the monospace font.

Colors

I personally like green and blue colors. That is what I use across the website wherever a touch of color is needed. The colors are from tailwindcss' palette. I might design a color scheme in future.
